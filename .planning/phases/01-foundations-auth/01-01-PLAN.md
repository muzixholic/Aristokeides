# Plan 01-01: Walking Skeleton & Auth API

## Objective
Establish the ASP.NET Core 9.0 Web API skeleton, configure Entity Framework Core with PostgreSQL, implement automatic migrations on startup, and build the JWT-based authentication pipeline.

<read_first>

<task>
<id>01-01-02</id>
<title>Database Setup and Auto-Migration</title>
<type>execute</type>
<read_first>
  - Aristokeides.Api/Program.cs
  - Aristokeides.Api/appsettings.json
</read_first>
<action>
1. Add `Npgsql.EntityFrameworkCore.PostgreSQL`, `Microsoft.EntityFrameworkCore.Design` packages to `Aristokeides.Api`.
2. Create `User` entity in `Models/User.cs` with `Id`, `Email`, `PasswordHash`, `Role`.
3. Create `AppDbContext` inheriting from `DbContext` with a `DbSet<User>`.
4. Add PostgreSQL connection string (`DefaultConnection`) to `appsettings.json` and configure `AppDbContext` in `Program.cs`.
5. Update `Program.cs` to call `context.Database.MigrateAsync()` automatically on startup inside a service scope.
</action>
<acceptance_criteria>
- `AppDbContext.cs` exists and registers `User`.
- `Program.cs` contains `await dbContext.Database.MigrateAsync()`.
- Connection string `DefaultConnection` is present in `appsettings.json`.
</acceptance_criteria>
</task>

<task>
<id>01-01-03</id>
<title>JWT Authentication and Controllers</title>
<type>execute</type>
<read_first>
  - Aristokeides.Api/Program.cs
  - Aristokeides.Api/Models/User.cs
</read_first>
<action>
1. Add `Microsoft.AspNetCore.Authentication.JwtBearer` package.
2. Add JWT configuration (Issuer, Audience, Key) to `appsettings.json` and register `AddJwtBearer` in `Program.cs`.
3. Ensure `Role` claim is baked into the JWT token upon generation.
4. Create `AuthController` with `POST /api/auth/register` (hashes password with `BCrypt.Net-Next`) and `POST /api/auth/login` (verifies and returns JWT).
5. Create `UsersController` with a protected `GET /api/users/me` endpoint (`[Authorize]`) that returns the current user's info based on the JWT claims.
6. Add Swagger support for JWT bearer tokens in `Program.cs`.
</action>
<acceptance_criteria>
- `AuthController.cs` contains `Register` and `Login` endpoints.
- `UsersController.cs` contains `[Authorize]` `Me` endpoint.
- `Program.cs` contains `builder.Services.AddAuthentication().AddJwtBearer(...)`.
</acceptance_criteria>
</task>

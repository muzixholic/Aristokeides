## Phase Goal

**As a** PR reviewer, **I want to** write inline comments on code lines in a PR diff and reply to threads, **so that** we can review and resolve discussions before merging.

<objective>
이 계획은 인라인 댓글 시스템에 마크다운 렌더링(XSS 방지 포함), 대댓글(답글) 작성 기능 및 대화 해결/재개 상태 토글 UI를 통합하여 코드 리뷰 스레드 기능을 완성합니다.
목적: 코드 리뷰 시 마크다운 서식을 활용하고, 토론을 스레드 형태로 전개하며, 해결된 논의를 정리하여 협업의 효율성을 높입니다.
산출물: MarkdownRenderer, PullRequestService의 답글/해결 로직, 탭 방식 마크다운 에디터 및 스레드 접기/해결 처리 UI.
</objective>

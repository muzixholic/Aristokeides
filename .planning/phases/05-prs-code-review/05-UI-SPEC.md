---

## Visual Hierarchy & Focal Points

- **Focal Point 1**: Pull Request 제목 및 상태 배지 (Open, Merged, Closed) - 최우선적으로 인지할 수 있도록 상단 강조
- **Focal Point 2**: Primary Action (Merge PR 버튼) - 우측 상단 또는 컨텍스트 중심에 Accent Color로 강조
- **Focal Point 3**: 코드 리뷰 Diff 및 코멘트 영역 - 본문에서 명확한 구분을 위해 Secondary 배경 사용

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, diff line number padding |
| sm | 8px | Compact element spacing, button padding |
| md | 16px | Default element spacing, container padding |
| lg | 24px | Section padding, header margin |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: none

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #ffffff | Background, surfaces, PR detail container |
| Secondary (30%) | #f3f4f6 | Diff file headers, tab backgrounds, secondary buttons |
| Accent (10%) | #2563eb | Create PR button, Merge PR button (if no conflict), Add Comment button, active tabs |
| Destructive | #ef4444 | Conflict warnings, error states, destructive actions |

Accent reserved for: Primary actions (Create PR, Merge PR, Add Comment), Active navigation tabs, and Links.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Create Pull Request, Merge Pull Request, Add Comment |
| Empty state heading | No Pull Requests Found |
| Empty state body | There are no pull requests yet. Create a new one to propose changes between branches. |
| Error state | Merge conflict detected. Please resolve conflicts locally before merging. |
| Destructive confirmation | Close Pull Request: Are you sure you want to close this pull request without merging? |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: none

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Background, surfaces (Default) |
| Secondary (30%) | #F3F4F6 | Cards, sidebar, nav, table headers (Default) |
| Accent (10%) | #2563EB | Links, Primary Buttons, Active Tabs (Default) |
| Destructive | #EF4444 | Destructive actions only (Default) |

Accent reserved for: 파일 및 디렉토리 링크, 커밋 해시(Commit hash), 활성화된 네비게이션 탭, 주요 액션 버튼

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | 커밋 내역 보기 (View Commits) / 파일 탐색 (Browse Files) |
| Empty state heading | 빈 저장소입니다 |
| Empty state body | 아직 커밋된 파일이 없습니다. Git 클라이언트를 사용하여 코드를 푸시해 주세요. |
| Error state | 항목을 찾을 수 없습니다. 브랜치나 경로가 올바른지 확인해 주세요. |
| Destructive confirmation | 해당 작업을 계속하시겠습니까? (Phase 3는 데이터 조회 위주이므로 파괴적 행동 없음) |

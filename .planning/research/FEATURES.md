# FEATURES.md

## Table Stakes (Must-haves)
- Git Repository Hosting (Git over HTTP/SSH)
- User Authentication & Authorization (Roles: Admin, Contributor, Reader)
- Issue Tracking (Basic CRUD, assignee, labels)
- Code Review / Pull Requests (Diff viewing, comments, merge strategies)
- Basic Webhooks (for integrations)

## Differentiators
- Extremely lightweight resource footprint (Gitea-like)
- Seamless C# / .NET ecosystem integration

## Anti-features (Do NOT build)
- Full CI/CD Runner Ecosystem (Out of scope for MVP)
- Kubernetes Cluster Management Features (Too heavy)
